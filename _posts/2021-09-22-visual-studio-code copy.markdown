## Purpose

* text editor with lots of plugin/extension
* features
  * git
* extension
  * project manager
  * markdown all in one
    * can format whole document (e.g. table)
  * markdownlint
* written using nodejs

## Troubleshoot

### Slow when loading or editing

* disable git auto refresh (Settings > Extension > Git > Autorefresh)
