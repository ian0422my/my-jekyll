## cheatsheet

* definition
  * array
    * kvp
    * object
* structure
  * same line
    * new kvp
  * new line
    * without identation
      * new kvp
    * with identation
      * with hyphen
        * new array
      * without hyphen
        * new object

```yaml
k1: 1
p1:
  p1p1: 1.1
  p1p2: 1.2
p2: 2
p3:
  - p3sp1: 3.1
  - p3sp2: 3.2
    p3sp3: 3.3
  - p3sp4: 
      p3sp4p1: 3.4.1
      p3sp4p2: 3.4.2
```

{
  "k1": 1,

    }

}
