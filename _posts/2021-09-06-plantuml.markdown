### Diagrams (What can be drawn?)

* activity
* class
* use case
* gannt chant
* object
* mindmap
* network
* sequence
* state
* timing
* work breakdown (wbs)
* wireframe (salt)

### Features (what diagram features is available?)

* comments
  * simple comment
  * block comment
* zoom
* title
* capture
* footer
* header
* legend

### how to draw 

#### on markdown

```markdown
{% plantuml %}
A->B: hello
B-A: go away!
{% endplantuml %}
```
