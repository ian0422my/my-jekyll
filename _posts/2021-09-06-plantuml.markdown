### Diagrams (What can be drawn?)

* [activity/flow chart](https://plantuml.com/activity-diagram-legacy)
* class
* use case
* gannt chant
* object
* [mindmap](https://plantuml.com/mindmap-diagram)
* [network](https://plantuml.com/nwdiag)
* [sequence](#sequence-diagram)
* state
* timing
* work breakdown (wbs)
* wireframe (salt)

### Features (what diagram features is available?)

* comments
  * simple comment
  * block comment
* zoom
* title
* capture
* footer
* header
* legend

### how to draw

#### on markdown

##### Sequence Diagram

```markdown
\{\% plantuml \%\}
A->B: hello12
B->A: go away!!!
* A
** AA
\{\% endplantuml \%\}
```

{% plantuml %}
+ A
++ AA
{% endplantuml %}
